# iPAS AI應用規劃師 初級 刷題平台

使用前請先參考網路開源文章，了解如何安排考前學習進度。

[如何在60日內取得AI應用規劃師初級證照（源自 Emily's LAB.）](https://emily50431.github.io/-emily-lu-website-g/blog/ipas-ai-planner-beginner/)

## Emily's LAB. 文章中提示詞
若考生依照文章中所提方式，使用Gemini進行測驗，可複製以下文字快速貼上提示詞。

```
1. 每組測驗結束，強制使用表格，將「我的選擇」、「正確答案」與「判定結果」並列，讓我一眼就能驗證。在輸出表格前，會針對我的原始文字進行二次掃描，確保不會再發生選項對錯行的情況。
2. 不要提供互動式測驗，遇到測驗題目都給我文字。
3. 回答時務必再三檢查，寧可回復時間久一點也不要出錯。
4. 請幫我記錄所有 IPAS AI 考試中答錯的題目與相關知識點，並在之後的練習中隨機抽考我。
5. 請給我實際的建議，不要過度吹捧，專注在專業與理性的角度做討論，不需要過度延伸，如果有需要我會繼續詢問。
```

## 本測驗說明
為方便考生進行刷題，本測驗使用 NotebookLM 之互動式測驗進行分享，與推薦文章所用方式不同，請視個人需求斟酌使用。右側面板有已建立之測驗以供使用，中間對話框可與AI討論相關問題或詢問相關知識。

### 公開測驗網址
由於資料內容包含世新大學資訊傳播學系 劉立民教授 所提供之課程檔案，因此本資源僅提供世新大學資訊傳播學系學生使用，為尊重智慧財產權請勿轉傳外流。特此感謝劉立民老師於世新111級資傳系班級提供相關學習資源。

[Notebook LM 測驗連結](https://notebooklm.google.com/notebook/1b924899-7090-488a-9100-0fbb8d89577c)

### 測驗種類說明
- 問題數量：皆為較多
- 難易度：依照系統分級分為「困難」、「中等」、「簡單」，可依測驗命名區分。

### 測驗製作方式說明
於本地儲存學習資料檔案 > 於 Notebook LM 建立新筆記本 > 於筆記本中左側上傳/匯入資料 > 於右側面板點選[測驗] > 選擇問題數量與難易度 > 輸入指定提示詞 > 等待生成完畢即完成步驟

本方式由世新大學資訊傳播學系學生 林辰禧 提供。

因系統與方式不同，因此內容與提示詞皆有所不同，若考生個人有額外資料，可模擬本方式制定專屬刷題系統。

## 本題庫提示詞
本提示詞由Gemini再製生成，歡迎取用或優化後再使用。

```
- 我要準備 IPAS AI 相關考試，請幫我根據上傳的來源資料製作複習用的互動式測驗
- 測驗必須以選擇題形式進行，每次只產生一道題目與選項，並等待我輸入答案後再進行結算，絕對不要一次給出多題或提前附上解答
- 在我回答後，強制使用表格將我的選擇、正確答案與判定結果並列，讓我一眼就能驗證
- 在輸出結算表格前，必須針對我的原始文字進行二次掃描比對，確保不會發生選項與對錯判定錯行的情況
- 請在對話過程中，持續幫我記錄所有答錯的題目與相關知識點，並在之後的練習中隨機抽考我
- 提供解析時請專注在專業與理性的角度做討論，給予實際的建議即可，不需要過度吹捧客套，也不需要過度延伸資料以外的內容
```
